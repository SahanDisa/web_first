Basic docuement structure

<html>
	<head>
		this tag contains all of the non-visual elements to help make the page work
	</head>
	<body>
		this tag defines the main content of the HTML docuement 
	</body>
</html>

--------------------------
Add a title

<html>
	<head>
		<title>First Page</title>
	</head>
	<body>
		<p>This is a paragraph</p> 
	</body>
</html>
